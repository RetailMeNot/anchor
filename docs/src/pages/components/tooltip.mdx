# Tooltip

The `Tooltip` component is used to display a hint to a user.

```jsx
import { Tooltip } from '@retailmenot/anchor';
```

<br />

```tsx live
<Tooltip content="Tooltip">
    <Button>Hover over me</Button>
</Tooltip>
```

<br />
