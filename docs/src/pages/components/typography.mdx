# Typography

The `Typography` component lets you easily add unified text styles to the page without having to create
a lot of extra CSS.

One of the principal features of this component is the concept of **`scale`**.
Although it's possible to alter the `line-height` (`lineHeight` prop) and `font-size` (`size` prop)
of `Typography`, it's recommended to alter its `scale` instead. Often, semantic HTML tagging
doesn't always get the same text styles and this helps alleviate that problem. It also assists in
rendering common fonts regardless of their enclosing tags.

Using `Typography` without modifying any of its props will wrap any text element in a `span` tag.
These are the default styles for Typography:

* `font-family: Avenir Next, Segoe UI, Helvetica Neue, Helvetica, Roboto,sans-serif;`
* `font-size: 1rem;`
* `line-height: 1.5rem;`
* `text-align: inherit;`
* `text-transform: none;`
* `color: inherit;`

<br />

```tsx live
<div>
    <Typography tag="p">
        The quick <Typography tag="span" transform="uppercase" color="#ff0000" weight="bolder">red</Typography> fox
    </Typography>
    <Typography tag="h2" color="charcoal" hue="light">
        jumped
    </Typography>
    <Typography tag="p">
        over the <Typography tag="span" scale="28" color="brown">brown</Typography> lazy dog.
    </Typography>
</div>
```
