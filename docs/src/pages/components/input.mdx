### Event Handlers

`Input` has its own methods tied to the various event handlers, `onBlur`, `onChange`, `onFocus`,
`onKeyDown`, and `onKeyUp`. This is intentional, as without these methods the generated `input`
element becomes an <a href="https://reactjs.org/docs/uncontrolled-components.html" target="_blank">uncontrolled component</a>.

The below example shows a way in which you can attach your own methods to these handlers.

```tsx live
<Component initialState={ {value: ''} }>
    {({ state, setState }) =>
        <Input
            label="Example Label"
            name="example-input"
            placeholder="Please Enter Text..."
            prefix={<ChevronRight />}
            size="md"
            suffix={<Question />}
            type="text"
            value={state.value}
            onChange={ newVal => setState(newVal) }
            onBlur={ newVal => console.log(newVal) }
        />
    }
</Component>
```
