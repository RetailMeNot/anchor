# Menu

<ComponentInfo title="Menu" />

`MenuItem` is a wrapper for the `a` tag, so any valid attribute for that element will be applied.

```tsx live
    <Menu
        background="primary.dark"
        color="white"
        justify="flex-start"
        size="lg"
    >
        <MenuItem label="RMN" href="//www.retailmenot.com" type="text/html" />
        <MenuItem label="GitHub" href="//www.github.com" as="a" target="_blank" />
        <MenuItem active label="Anchor" href="//anchor.retailmenot.design/" />
    </Menu>
```
