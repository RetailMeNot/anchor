# Menu

The `Menu` component pulls together links into a single horizontal menu bar with unified styling. It
is composed of two components, `Menu` itself, and its child elements, `MenuItem`. `MenuItem` is a
wrapper for the `a` tag, so any valid attribute for that element will be applied.

```tsx live
    <Menu
        background={colors.savvyCyan.base}
        color={colors.white.base}
        justify="flex-start"
        size="large"
    >
        <MenuItem label="RMN" href="//www.retailmenot.com" type="text/html" />
        <MenuItem label="GitHub" href="//www.github.com" as="a" target="_blank" />
        <MenuItem label="Anchor" href="//anchor.retailmenot.design/" active={true} />
    </Menu>
```

            </td>
            <td>String</td>
            <td>-</td>
        </tr>

            <td>String</td>
            <td>-</td>

            <td>-</td>
        </tr>
