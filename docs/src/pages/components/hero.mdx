# Hero

<ComponentInfo title="Hero" />

Aside from the `Hero` component itself, it has two additional components that can be deconstructed
from it.

```jsx
    import { Hero } from '@retailmenot/anchor';

    const { Title, Subtitle } = Hero;
```

Depending on how your environment is set up, you could directly access these sub-components directly
in your jsx as shown below. This example uses the <Link to="/theme/colors/">`colors`</Link> export
from **Anchor** as well as the <Link to="/components/button/">`Button`</Link> component.

```tsx live
    <Hero
        background="primary.dark"
        minHeight="12.5rem"
    >
        <Hero.Title weight="bolder">Hero Title!</Hero.Title>
        <Hero.Subtitle scale="16" mb="0.5rem">Hero Subtitle!</Hero.Subtitle>

        <div>
            <Button
                variant="outline"
                reverse
                onClick={() => alert('Hello!')}
            >
                A Button
            </Button>
        </div>
    </Hero>
```

`Title` is as a wrapper of the `Typography` component.
