# Button

<ComponentInfo title="Button" mode="description" />

*NOTE! The example in the live code editor below is using the default `variant`, `filled`.
Editing this field can cause the live code editor to crash, so examples of those variants are provided
below. Also included is an example of an icon button*

```tsx live
<Component initialState={{ size: 'md' }}>
  {({ state, setState }) =>
    <div>
      <Button
        block={false}
        circular={false}
        colorTheme={colors.savvyCyan}
        disabled={false}
        flip={false}
        forceActive={false}
        forceFocus={false}
        forceHover={false}
        minWidth="20rem"
        onClick={() => setState(({ size }) => ({
          size: size === 'sm' ? 'md' : 'sm'
        }))}
        outline={true}
        prefix={<AddEvent />}
        reverse={false}
        size={state.size}
        suffix={false}
      >
        Button Text
      </Button>
    </div>
  }
</Component>
```

### Variants

###### Filled Button
```tsx live hideTitle
<Component initialState={{ size: 'md' }}>
  {({ state, setState }) =>
    <div>
      <Button
        onClick={() => setState(({ size }) => ({
          size: size === 'md' ? 'xs' : 'md'
        }))}
        size={state.size}
        variant="filled"
      >
        I'm size '{state.size}'!
      </Button>
    </div>
  }
</Component>
```
