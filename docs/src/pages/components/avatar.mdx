# Avatar

The `Avatar` component renders a generic avatar graphic, or a stylized label.

```tsx live
    <section>
        <div>
            <Typography tag="h3">Avatar with a label...</Typography>
            <Avatar label="ps" />
        </div>

        <div>
            <Typography tag="h3">Avatar without a label...</Typography>
            <Avatar />
        </div>
    </section>
```
