# ColorBlurb

Many of the components have props for colors, and most of them have very similar descriptions
in the api table. This component tries to consolidate variations of those descriptions in one
place.

###### ColorBlurb, Basic
```tsx live hideTitle
<ColorBlurb label="Example Component" />
```

###### ColorBlurb, background color
```tsx live hideTitle
<ColorBlurb background label="Example Component" />
```

###### ColorBlurb, background color & gradients
```tsx live hideTitle
<ColorBlurb background gradient label="Example Component" />
```

###### ColorBlurb, theme colors
```tsx live hideTitle
<ColorBlurb defaultTheme gradient label="Example Component" />
```
